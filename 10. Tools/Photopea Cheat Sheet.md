## 🖱️ Navigation & Hold Actions

### Scroll & Zoom Controls

- **Vertical Scroll:** `Wheel Up` / `Wheel Down`
    
- **Horizontal Scroll:** `Ctrl` + `Wheel Up` / `Wheel Down`
    
- **Zooming:** `Alt` + `Wheel Up` / `Wheel Down`
    

### Quick Tools (Hold to Use, Release to Go Back)

- **`Ctrl`** ➔ Hold to temporary enable the **Move tool**.
    
- **`Space`** ➔ Hold to temporary enable the **Hand tool** (pan canvas).
    
- **`Ctrl + Space`** ➔ Hold to temporary enable the **Zoom tool**.
    

## 🛠️ Tool Palette (Single-Key Shortcuts)

Press these corresponding letters to select tools. Repeated presses or hidden menus navigate sub-tools tied to the same hotkey.

Code snippet

```
graph TD
    V[V: Move] --> M[M: Marquee Select] --> L[L: Lasso Pack]
    L --> W[W: Wand / Quick Select] --> C[C: Crop Pack] --> I[I: Eyedropper]
    I --> J[J: Healing / Patch] --> B[B: Brush / Pencil] --> S[S: Clone Stamp]
    S --> E[E: Erasers] --> G[G: Gradient / Bucket] --> O[O: Dodge / Burn]
    O --> T[T: Text Tools] --> P[P: Pen Vector Pack] --> A[A: Path Selection]
    A --> U[U: Vector Shapes] --> H[H: Hand / Rotate] --> Z[Z: Zoom]
```

|**Key**|**Tool Assignments**|
|---|---|
|**`V`**|Move Tool|
|**`M`**|Rectangle Select \| Ellipse Select|
|**`L`**|Lasso Select \| Polygonal Lasso Select \| Magnetic Lasso Select|
|**`W`**|Magic Wand \| Quick Selection \| Object Selection|
|**`C`**|Crop Tool \| Perspective Crop \| Slice Tool \| Slice Select Tool|
|**`I`**|Eyedropper \| Color Sampler \| Ruler|
|**`J`**|Healing Brush Tool \| Magic Replace \| Patch Tool \| Content-Aware Move Tool \| Red Eye Tool|
|**`B`**|Brush Tool \| Pencil Tool \| Color Replacement|
|**`S`**|Clone Tool|
|**`E`**|Eraser Tool \| Background Eraser \| Magic Eraser|
|**`G`**|Gradient Tool \| Paint Bucket Tool|
|**`O`**|Dodge Tool \| Burn Tool \| Sponge Tool|
|**`T`**|Type Tool \| Vertical Type Tool|
|**`P`**|Pen \| Free Pen \| Curvature Pen \| Add/Delete Anchor Point \| Convert Point|
|**`A`**|Path Select \| Direct Select|
|**`U`**|Rectangle \| Ellipse \| Line \| Parametric Shape \| Custom Shape|
|**`H`**|Hand Tool \| Rotate View|
|**`Z`**|Zoom Tool|
