## 🎛️ Track Control Shortcuts (Edit Cont.)

### Track Destination Selection

- **Video Destinations (V1–V8):** `Option` + `[1-8]`
    
- **Audio Destinations (A1–A4):** `Option` + `Command` + `[1-4]`
    
- **Audio Destinations (A5–A8):** `Option` + `Command` + `[5-8]`
    

### Lock / Unlock Tracks

- **Video Tracks (V1–V8):** `Option` + `Shift` + `[1-8]`
    
- **Lock/Unlock All Video:** `Option` + `Shift` + `9`
    
- **Audio Tracks (A1–A8):** `Option` + `Shift` + `[F1-F8]`
    
- **Lock/Unlock All Audio:** `Option` + `Shift` + `F9`
    

### Auto Track Selector

- **Video Auto-Select (V1–V8):** `Option` + `[F1-F8]`
    
- **Auto-Select All Video:** `Option` + `F9`
    
- **Audio Auto-Select (A1–A4):** `Option` + `Command` + `[F1-F4]`
    
- **Audio Auto-Select (A5–A8):** `Option` + `Command` + `[F5-F8]`
    
- **Auto-Select All Audio:** `Option` + `Command` + `F9`
