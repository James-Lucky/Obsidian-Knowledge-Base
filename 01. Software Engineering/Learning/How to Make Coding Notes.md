## 2. Algorithm / Problem Notes  
  
For every problem, write:  
  
- [ ] Logic  
- [ ] Code / Pseudocode  
- [ ] Tips & Tricks  
- [ ] Time & Space Complexity

## 3. Use Both Types of Notes  
  
- 📒 Physical Notebook  
- 💻 Digital Notes (Obsidian/Laptop)  
  
> [!tip]  
> Handwritten notes improve retention, while digital notes make searching and revising much easier.
