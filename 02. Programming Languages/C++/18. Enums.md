# C++ 

## What is an Enum?

An `enum` (Enumeration) is a user-defined data type that allows you to assign names to a set of integral constants.

Enums make code more readable and maintainable.

---

## Syntax

```cpp
enum EnumName {
    VALUE1,
    VALUE2,
    VALUE3
};
```

---

## Example

```cpp
enum Day {
    MONDAY,
    TUESDAY,
    WEDNESDAY
};
```

By default:

```cpp
MONDAY    = 0
TUESDAY   = 1
WEDNESDAY = 2
```

---

## Using Enums

```cpp
Day today = TUESDAY;

std::cout << today;
```

### Output

```text
1
```

> [!note]  
> Enums store integer values internally.

---

## Assigning Custom Values

```cpp
enum Status {
    SUCCESS = 200,
    NOT_FOUND = 404,
    SERVER_ERROR = 500
};
```

### Example

```cpp
std::cout << SUCCESS;
```

### Output

```text
200
```

---

## Automatic Value Assignment

```cpp
enum Numbers {
    ONE = 1,
    TWO,
    THREE
};
```

Values become:

```text
ONE   = 1
TWO   = 2
THREE = 3
```

---

## Enum in Switch Statements

```cpp
enum Day {
    MONDAY,
    TUESDAY,
    WEDNESDAY
};

Day today = MONDAY;

switch (today) {
    case MONDAY:
        std::cout << "Monday";
        break;

    case TUESDAY:
        std::cout << "Tuesday";
        break;
}
```

## Scoped Enums (`enum class`)

Introduced in C++11.

```cpp
enum class Color {
    RED,
    GREEN,
    BLUE
};
```

Usage:

```cpp
Color color = Color::RED;
```

> [!important]  
> `enum class` is safer and avoids name conflicts.

---

## Enum vs Enum Class

|`enum`|`enum class`|
|---|---|
|Implicit conversion to `int`|No implicit conversion|
|Global scope|Scoped|
|Less safe|More safe|

### Example

```cpp
enum Color { RED };
enum class Status { RED };
```

`enum class` prevents naming collisions.

---

## Important Points

- Enums represent named integer constants.
    
- Default values start from `0`.
    
- Custom values can be assigned.
    
- `enum class` is preferred in modern C++.
    

---

## Common Mistakes

- Forgetting the semicolon after an enum definition.
    
- Assuming enums store strings.
    
- Using regular enums when `enum class` is more appropriate.
    
- Expecting `enum class` values to convert automatically to integers.
    

---

## Extra Information

### Converting `enum class` to Integer

```cpp
enum class Color {
    RED
};

int value = static_cast<int>(Color::RED);
```

This is required because `enum class` does not implicitly convert to `int`.
