# C++ 

## What is a Boolean?

A Boolean is a data type that can store only two values:

- `true`
    
- `false`
    

Boolean values are represented using the `bool` data type.

---

## Declaring Boolean Variables

### Syntax

```cpp
bool variableName = value;
```

### Example

```cpp
bool isLoggedIn = true;
bool isAdmin = false;
```

---

## Boolean Values

```cpp
std::cout << true;
std::cout << false;
```

### Output

```text
1
0
```

> [!note]  
> Internally:
> 
> - `true` = `1`
>     
> - `false` = `0`
>     

---

## Using Booleans with Comparisons

Comparison operators return Boolean values.

```cpp
std::cout << (10 > 5);
```

### Output

```text
1
```

---

### Examples

```cpp
std::cout << (10 == 10); // true
std::cout << (10 != 10); // false
std::cout << (5 < 3);    // false
```

---

## Boolean Expressions

```cpp
int age = 21;

bool canVote = (age >= 18);
```

---

## Using Booleans in Conditions

```cpp
bool isStudent = true;

if (isStudent) {
    std::cout << "Discount Applied";
}
```

---

## Logical Operators

|Operator|Description|
|---|---|
|`&&`|Logical AND|
|`||
|`!`|Logical NOT|

### Example

```cpp
bool a = true;
bool b = false;

std::cout << (a && b);
```

### Output

```text
0
```

---

## Truthy and Falsy Values

C++ treats some values as `true` and others as `false`.

|Value|Boolean Equivalent|
|---|---|
|`0`|`false`|
|Non-zero|`true`|

### Example

```cpp
if (100) {
    std::cout << "Executed";
}
```

The code executes because `100` is considered `true`.

---

## Example Program

```cpp
#include <iostream>

int main() {
    int marks = 85;

    bool passed = (marks >= 40);

    std::cout << passed;

    return 0;
}
```

### Output

```text
1
```

---

## Important Points

- `bool` stores only `true` or `false`.
    
- `true` is represented as `1`.
    
- `false` is represented as `0`.
    
- Comparison operations return Boolean values.
    
- Non-zero values are treated as `true`.
    

---

## Common Mistakes

- Writing `True` or `False` instead of `true` and `false`.
    
- Confusing `=` with `==`.
    
- Assuming only `1` is considered `true`.
    
- Forgetting that `0` evaluates to `false`.
    

---

## Extra Information

### Displaying `true` and `false`

By default:

```cpp
std::cout << true;
```

Output:

```text
1
```

To display actual words:

```cpp
std::cout << std::boolalpha << true;
```

Output:

```text
true
```
