# C++ Pointers

## What is a Pointer?

A pointer is a variable that stores the memory address of another variable.

Pointers provide direct access to memory and are one of the most powerful features of C++.

---

## Syntax

```cpp
dataType* pointerName;
```

### Example

```cpp
int* ptr;
```

---

## Address Operator (`&`)

The `&` operator returns the memory address of a variable.

```cpp
int number = 10;

std::cout << &number;
```

### Output

```text
0x61ff08
```

> [!note]  
> Memory addresses vary every time the program runs.

---

## Declaring and Initializing Pointers

```cpp
int number = 10;

int* ptr = &number;
```

Memory Representation:

```text
number = 10
ptr    = Address of number
```

---

## Dereference Operator (`*`)

The `*` operator is used to access the value stored at a memory address.

```cpp
int number = 10;
int* ptr = &number;

std::cout << *ptr;
```

### Output

```text
10
```

---

## Modifying Values Using Pointers

```cpp
int number = 10;
int* ptr = &number;

*ptr = 50;

std::cout << number;
```

### Output

```text
50
```

---

## Pointer and Address

```cpp
int x = 25;
int* ptr = &x;

std::cout << ptr << "\n";
std::cout << *ptr;
```

### Output (Example)

```text
0x61ff08
25
```

---

## Null Pointers

A null pointer points to nothing.

```cpp
int* ptr = nullptr;
```

> [!important]  
> Always initialize pointers. Use `nullptr` when there is no valid address.

---

## Pointer Arithmetic

```cpp
int arr[] = {10, 20, 30};

int* ptr = arr;

std::cout << *(ptr + 1);
```

### Output

```text
20
```

Pointer arithmetic moves according to the size of the data type.

---

## Pointers and Arrays

```cpp
int arr[] = {1, 2, 3};

std::cout << arr[0];
std::cout << *(arr);
```

Both produce the same output.

---

## Pointer to Pointer

```cpp
int x = 10;

int* ptr = &x;
int** ptr2 = &ptr;
```

Memory Representation:

```text
x    -> 10
ptr  -> Address of x
ptr2 -> Address of ptr
```

---

## Constant Pointers

### Pointer to Constant

```cpp
const int* ptr;
```

- Value cannot be changed.
    
- Pointer can point elsewhere.
    

---

### Constant Pointer

```cpp
int* const ptr = &x;
```

- Pointer cannot point elsewhere.
    
- Value can be changed.
    

---

### Constant Pointer to Constant

```cpp
const int* const ptr = &x;
```

- Neither the pointer nor the value can change.
    

---

## Dynamic Memory Allocation

```cpp
int* ptr = new int(100);

delete ptr;
```

`new` allocates memory.  
`delete` releases memory.

> [!important]  
> Forgetting `delete` causes memory leaks.

---

## References vs Pointers

|References|Pointers|
|---|---|
|Cannot be null|Can be null|
|Simpler|More flexible|
|Must be initialized|Optional initialization|
|No pointer arithmetic|Supports pointer arithmetic|

---

## Important Points

- Pointers store memory addresses.
    
- `&` gives an address.
    
- `*` dereferences a pointer.
    
- Always initialize pointers.
    
- Use `nullptr` instead of `NULL` in modern C++.
    

---

## Common Mistakes

- Dereferencing a null pointer.
    

```cpp
int* ptr = nullptr;

std::cout << *ptr; // Undefined Behavior
```

- Forgetting `delete`.
    
- Using uninitialized pointers.
    
- Accessing invalid memory locations.
    

---

## Extra Information

### Smart Pointers (Modern C++)

Modern C++ prefers:

- `std::unique_ptr`
    
- `std::shared_ptr`
    
- `std::weak_ptr`
    

```cpp
#include <memory>

std::unique_ptr<int> ptr =
    std::make_unique<int>(10);
```

Smart pointers automatically manage memory and help prevent memory leaks.
