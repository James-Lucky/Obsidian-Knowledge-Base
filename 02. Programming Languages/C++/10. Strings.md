# C++ 

## What are Strings?

A string is a sequence of characters used to store text.

Examples:

- `"Hello"`
    
- `"Lucky"`
    
- `"C++ Programming"`
    

In C++, strings are provided by the `<string>` library.

---

## Including the String Library

```cpp
#include <string>
```

---

## Declaring a String

### Syntax

```cpp
std::string variableName = "value";
```

### Example

```cpp
std::string name = "Lucky";
```

---

## Printing Strings

```cpp
#include <iostream>
#include <string>

int main() {
    std::string language = "C++";

    std::cout << language;
}
```

### Output

```text
C++
```

---

## String Concatenation

### Using `+`

```cpp
std::string firstName = "Lucky";
std::string lastName = "Sharma";

std::string fullName = firstName + " " + lastName;
```

### Output

```text
Lucky Sharma
```

---

## Using `append()`

```cpp
std::string str1 = "Hello ";
std::string str2 = "World";

str1.append(str2);
```

---

## String Length

### `length()`

```cpp
std::string text = "Programming";

std::cout << text.length();
```

### `size()`

```cpp
std::cout << text.size();
```

> [!note]  
> `length()` and `size()` return the same value.

---

## Accessing Characters

Strings use zero-based indexing.

```cpp
std::string name = "Lucky";

std::cout << name[0];
```

### Output

```text
L
```

---

## Modifying Characters

```cpp
std::string name = "Lucky";

name[0] = 'B';
```

### Output

```text
Bucky
```

---

## User Input with Strings

### Using `cin`

```cpp
std::string name;

std::cin >> name;
```

`cin` reads only until the first space.

---

### Using `getline()`

```cpp
std::string fullName;

getline(std::cin, fullName);
```

This reads the entire line, including spaces.

---

## Escape Characters

|Escape Character|Description|
|---|---|
|`\n`|New Line|
|`\t`|Tab|
|`\"`|Double Quote|
|`\\`|Backslash|

### Example

```cpp
std::cout << "Hello\nWorld";
```

---

## Common String Functions

|Function|Description|
|---|---|
|`length()`|Returns string length|
|`size()`|Returns string length|
|`append()`|Adds text to the end|
|`substr()`|Returns a substring|
|`find()`|Finds a character/string|
|`empty()`|Checks if string is empty|
|`clear()`|Removes all characters|

### Example

```cpp
std::string str = "Programming";

std::cout << str.substr(0, 4);
```

### Output

```text
Prog
```

---

## Important Points

- Strings are objects of the `std::string` class.
    
- String indexing starts at `0`.
    
- `getline()` is preferred for multi-word input.
    
- `length()` and `size()` are equivalent.
    

---

## Common Mistakes

- Forgetting `#include <string>`.
    
- Using `cin` for full names.
    
- Accessing an invalid index.
    
- Using double quotes (`"A"`) instead of single quotes (`'A'`) for characters.
    

---

## Extra Information

### C-Style Strings

Before `std::string`, C++ used character arrays.

```cpp
char name[] = "Lucky";
```

Modern C++ prefers:

```cpp
std::string name = "Lucky";
```

> [!important]  
> Use `std::string` whenever possible. It is safer, easier to use, and provides many built-in functions.
