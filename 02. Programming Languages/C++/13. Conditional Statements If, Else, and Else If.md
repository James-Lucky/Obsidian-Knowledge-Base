# C++ 

## What are Conditional Statements?

Conditional statements allow a program to make decisions based on whether a condition is `true` or `false`.

---

## Types of Conditional Statements

1. `if`
    
2. `else`
    
3. `else if`
    
4. Nested `if`
    
5. Short-hand `if` (Ternary Operator)
    

---

## `if` Statement

Executes a block of code only if the condition is `true`.

### Syntax

```cpp
if (condition) {
    // Code to execute
}
```

### Example

```cpp
int age = 20;

if (age >= 18) {
    std::cout << "You can vote.";
}
```

---

## `else` Statement

Executes when the `if` condition is `false`.

### Syntax

```cpp
if (condition) {
    // Code
} else {
    // Code
}
```

### Example

```cpp
int age = 16;

if (age >= 18) {
    std::cout << "Adult";
} else {
    std::cout << "Minor";
}
```

---

## `else if` Statement

Used to check multiple conditions.

### Syntax

```cpp
if (condition1) {
    // Code
}
else if (condition2) {
    // Code
}
else {
    // Code
}
```

### Example

```cpp
int marks = 85;

if (marks >= 90) {
    std::cout << "Grade A";
}
else if (marks >= 75) {
    std::cout << "Grade B";
}
else {
    std::cout << "Grade C";
}
```

---

## Nested `if`

An `if` statement inside another `if` statement.

```cpp
int age = 20;
bool hasID = true;

if (age >= 18) {
    if (hasID) {
        std::cout << "Entry Allowed";
    }
}
```

---

## Logical Operators with Conditions

### AND (`&&`)

```cpp
if (age >= 18 && hasID) {
    std::cout << "Access Granted";
}
```

### OR (`||`)

```cpp
if (age < 18 || hasPermission) {
    std::cout << "Allowed";
}
```

### NOT (`!`)

```cpp
if (!isLoggedIn) {
    std::cout << "Please Login";
}
```

---

## Short-Hand `if` (Ternary Operator)

### Syntax

```cpp
condition ? expression1 : expression2;
```

### Example

```cpp
int age = 18;

std::string result =
    (age >= 18) ? "Adult" : "Minor";
```

---

## Example Program

```cpp
#include <iostream>

int main() {
    int number = 10;

    if (number > 0) {
        std::cout << "Positive";
    } else if (number < 0) {
        std::cout << "Negative";
    } else {
        std::cout << "Zero";
    }

    return 0;
}
```

---

## Important Points

- Conditions must evaluate to `true` or `false`.
    
- Only one block in an `if-else if-else` chain executes.
    
- Curly braces `{}` improve readability.
    
- Nested `if` statements should be used carefully.
    

---

## Common Mistakes

- Using `=` instead of `==`.
    
- Forgetting curly braces.
    
- Writing overlapping conditions.
    
- Making nested conditions unnecessarily complex.
    

---

## Extra Information

### Condition Evaluation Order

```cpp
if (x > 10) {
    // Checked first
}
else if (x > 5) {
    // Checked second
}
```

Conditions are evaluated from top to bottom. Once a condition is `true`, the remaining conditions are skipped.

> [!important]  
> Always place more specific conditions before general ones.
