# C++ While Loop

## What is a While Loop?

A `while` loop repeatedly executes a block of code as long as a condition is `true`.

It is useful when the number of iterations is unknown beforehand.

---

## Syntax

```cpp
while (condition) {
    // Code to execute
}
```

---

## Example

```cpp
int i = 1;

while (i <= 5) {
    std::cout << i << " ";
    i++;
}
```

### Output

```text
1 2 3 4 5
```

---

## How It Works

1. Check the condition.
    
2. If `true`, execute the loop body.
    
3. Update the loop variable.
    
4. Repeat until the condition becomes `false`.
    

---

## Infinite Loop

If the condition never becomes `false`, the loop runs forever.

```cpp
while (true) {
    std::cout << "Infinite Loop";
}
```

> [!important]  
> Always ensure the loop condition eventually becomes `false`.

---

## Countdown Example

```cpp
int count = 5;

while (count > 0) {
    std::cout << count << "\n";
    count--;
}
```

### Output

```text
5
4
3
2
1
```

---

## User Input Example

```cpp
int number;

std::cin >> number;

while (number != 0) {
    std::cout << number << "\n";
    std::cin >> number;
}
```

The loop continues until the user enters `0`.

---

## Nested While Loop

```cpp
int i = 1;

while (i <= 3) {
    int j = 1;

    while (j <= 2) {
        std::cout << "* ";
        j++;
    }

    std::cout << "\n";
    i++;
}
```

### Output

```text
* *
* *
* *
```

---

## Important Points

- `while` checks the condition before execution.
    
- The loop body may execute zero times.
    
- Always update the loop variable.
    
- Nested loops are allowed.
    

---

## Common Mistakes

- Forgetting to update the loop variable.
    
- Creating unintended infinite loops.
    
- Writing incorrect conditions.
    
- Modifying the wrong variable inside the loop.
    

---

## Extra Information

### `while` vs `for`

|While Loop|For Loop|
|---|---|
|Unknown iterations|Known iterations|
|Condition-focused|Counter-focused|

```cpp
while (condition) {
    // Code
}
```

```cpp
for (int i = 0; i < 10; i++) {
    // Code
}
```

Use `while` when you don't know how many times the loop should run.

# C++ Do While Loop

## What is a Do While Loop?

A `do while` loop executes a block of code at least once, regardless of whether the condition is `true` or `false`.

Unlike the `while` loop, the condition is checked after the loop body executes.

---

## Syntax

```cpp
do {
    // Code to execute
} while (condition);
```

> [!important]  
> Notice the semicolon (`;`) after the `while(condition)` statement.

---

## Example

```cpp
int i = 1;

do {
    std::cout << i << " ";
    i++;
} while (i <= 5);
```

### Output

```text
1 2 3 4 5
```

---

## How It Works

1. Execute the loop body.
    
2. Check the condition.
    
3. If `true`, repeat.
    
4. If `false`, exit the loop.
    

---

## Difference Between `while` and `do while`

### While Loop

```cpp
int x = 10;

while (x < 5) {
    std::cout << "Hello";
}
```

### Output

```text
(No Output)
```

---

### Do While Loop

```cpp
int x = 10;

do {
    std::cout << "Hello";
} while (x < 5);
```

### Output

```text
Hello
```

> [!note]  
> A `do while` loop always runs at least once.

---

## User Input Example

```cpp
int number;

do {
    std::cout << "Enter a number (0 to exit): ";
    std::cin >> number;
} while (number != 0);
```

---

## Menu Example

```cpp
int choice;

do {
    std::cout << "\n1. Start";
    std::cout << "\n2. Exit";
    std::cout << "\nEnter Choice: ";

    std::cin >> choice;

} while (choice != 2);
```

This is a common use case for `do while` loops.

---

## Nested Do While Loop

```cpp
int i = 1;

do {
    int j = 1;

    do {
        std::cout << "* ";
        j++;
    } while (j <= 3);

    std::cout << "\n";
    i++;
} while (i <= 2);
```

### Output

```text
* * *
* * *
```

---

## Important Points

- The loop body executes before checking the condition.
    
- A `do while` loop executes at least once.
    
- A semicolon is required after `while(condition)`.
    
- Useful for menus and user input.
    

---

## Common Mistakes

- Forgetting the semicolon after `while(condition)`.
    
- Using `do while` when a regular `while` loop is more appropriate.
    
- Creating infinite loops by not updating variables.
    

---

## Extra Information

### When to Use `do while`

Use `do while` when:

- The code must execute at least once.
    
- Building menu-driven programs.
    
- Taking user input repeatedly.
    
- Creating retry mechanisms.
    

```cpp
do {
    // Execute once
} while (condition);
```

# C++ For Loop

## What is a For Loop?

A `for` loop is used to execute a block of code a specific number of times.

It is commonly used when the number of iterations is known in advance.

---

## Syntax

```cpp
for (initialization; condition; update) {
    // Code to execute
}
```

### Components

- **Initialization:** Executes once at the beginning.
    
- **Condition:** Checked before each iteration.
    
- **Update:** Executes after each iteration.
    

---

## Example

```cpp
for (int i = 1; i <= 5; i++) {
    std::cout << i << " ";
}
```

### Output

```text
1 2 3 4 5
```

---

## How It Works

```cpp
for (int i = 1; i <= 3; i++) {
    std::cout << i;
}
```

Execution Flow:

1. `int i = 1`
    
2. Check `i <= 3`
    
3. Execute loop body.
    
4. Execute `i++`
    
5. Repeat until the condition becomes `false`.
    

---

## Countdown Example

```cpp
for (int i = 5; i >= 1; i--) {
    std::cout << i << "\n";
}
```

### Output

```text
5
4
3
2
1
```

---

## Increment by Different Values

```cpp
for (int i = 0; i <= 10; i += 2) {
    std::cout << i << " ";
}
```

### Output

```text
0 2 4 6 8 10
```

---

## Nested For Loop

```cpp
for (int i = 1; i <= 3; i++) {
    for (int j = 1; j <= 2; j++) {
        std::cout << "* ";
    }

    std::cout << "\n";
}
```

### Output

```text
* *
* *
* *
```

---

## Infinite For Loop

```cpp
for (;;) {
    std::cout << "Running...";
}
```

> [!important]  
> `for(;;)` creates an infinite loop.

---

## Range-Based For Loop (C++11)

Used to iterate over arrays and containers.

```cpp
int numbers[] = {1, 2, 3, 4};

for (int num : numbers) {
    std::cout << num << " ";
}
```

### Output

```text
1 2 3 4
```

> [!note]  
> Range-based `for` loops are heavily used with STL containers like `vector`.

---

## Important Points

- Use `for` when the number of iterations is known.
    
- Initialization, condition, and update are optional.
    
- Nested `for` loops are common in pattern problems.
    
- `for(;;)` creates an infinite loop.
    

---

## Common Mistakes

- Writing incorrect loop conditions.
    
- Forgetting to update the loop variable.
    
- Creating off-by-one errors.
    
- Using `<=` when `<` is required.
    

---

## Extra Information

### `while` vs `for`

|While Loop|For Loop|
|---|---|
|Unknown iterations|Known iterations|
|Condition-based|Counter-based|
|More flexible|More concise|

```cpp
for (int i = 0; i < n; i++) {
    // Preferred for counting loops
}
```

In competitive programming and DSA, the `for` loop is the most frequently used loop.

# C++ 

## What are `break` and `continue`?

`break` and `continue` are control statements used to alter the normal flow of loops and `switch` statements.

- `break` → Terminates the loop or `switch`.
    
- `continue` → Skips the current iteration and moves to the next one.
    

---

## `break` Statement

### Syntax

```cpp
break;
```

### Example

```cpp
for (int i = 1; i <= 10; i++) {
    if (i == 5) {
        break;
    }

    std::cout << i << " ";
}
```

### Output

```text
1 2 3 4
```

> [!important]  
> Once `break` executes, the loop ends immediately.

---

## `break` with `while`

```cpp
int i = 1;

while (true) {
    if (i == 4) {
        break;
    }

    std::cout << i << " ";
    i++;
}
```

### Output

```text
1 2 3
```

---

## `break` in `switch`

```cpp
switch (choice) {
    case 1:
        std::cout << "Start";
        break;

    case 2:
        std::cout << "Exit";
        break;
}
```

Without `break`, fall-through occurs.

---

## `continue` Statement

### Syntax

```cpp
continue;
```

### Example

```cpp
for (int i = 1; i <= 5; i++) {
    if (i == 3) {
        continue;
    }

    std::cout << i << " ";
}
```

### Output

```text
1 2 4 5
```

> [!note]  
> `continue` skips only the current iteration; the loop continues executing.

---

## `continue` with `while`

```cpp
int i = 0;

while (i < 5) {
    i++;

    if (i == 3) {
        continue;
    }

    std::cout << i << " ";
}
```

### Output

```text
1 2 4 5
```

---

## Difference Between `break` and `continue`

|`break`|`continue`|
|---|---|
|Exits the loop completely|Skips current iteration|
|Used in loops and `switch`|Used only in loops|
|Stops execution immediately|Continues with the next iteration|

---

## Nested Loop Example

```cpp
for (int i = 1; i <= 3; i++) {
    for (int j = 1; j <= 3; j++) {
        if (j == 2) {
            break;
        }

        std::cout << j << " ";
    }

    std::cout << "\n";
}
```

### Output

```text
1
1
1
```

> [!important]  
> `break` only exits the innermost loop.

---

## Important Points

- `break` immediately terminates the current loop or `switch`.
    
- `continue` skips the current iteration.
    
- `break` affects only the innermost loop in nested loops.
    
- `continue` does not stop loop execution.
    

---

## Common Mistakes

- Expecting `break` to exit all nested loops.
    
- Forgetting to update variables before `continue` in `while` loops.
    
- Confusing `break` and `continue`.
    
- Using `continue` when `break` is required.
    

---

## Extra Information

### Exiting Multiple Loops

C++ does not provide a direct way to break out of multiple nested loops.

Common approaches:

- Use flags.
    
- Use functions and `return`.
    
- Use `goto` (generally discouraged).
    

```cpp
bool found = false;
```
