# C++ Syntax

## What is Syntax?

Syntax refers to the set of rules that define how C++ code must be written. If the syntax is incorrect, the program will produce errors and will not compile.

---

## Basic Syntax Structure

```cpp
#include <iostream>

int main() {
    std::cout << "Hello, World!";
    return 0;
}
```

---

## Breakdown of Syntax

### Header File

```cpp
#include <iostream>
```

- Includes the Input/Output stream library.
    
- Allows the use of `std::cout` and `std::cin`.
    

---

### Main Function

```cpp
int main() {
    // Code goes here
}
```

- Every C++ program starts execution from `main()`.
    
- It must return an integer value.
    

---

### Statements

A statement is an instruction given to the compiler.

```cpp
int age = 21;
std::cout << age;
```

Each statement usually ends with a semicolon (`;`).

---

### Semicolon (`;`)

```cpp
int x = 10;
```

- Indicates the end of a statement.
    
- Missing a semicolon results in a compilation error.
    

---

### Curly Braces `{}`

```cpp
{
    // Block of code
}
```

- Used to define a block or scope.
    
- Commonly used in functions, loops, and conditionals.
    

---

### Whitespace

C++ ignores extra spaces, tabs, and new lines.

```cpp
int x = 10;
```

```cpp
int     x     =     10;
```

Both are valid.

---

## Code Formatting

Good:

```cpp
int main() {
    int age = 21;
    std::cout << age;
    return 0;
}
```

Bad:

```cpp
int main(){int age=21;std::cout<<age;return 0;}
```

> [!tip]  
> Proper formatting improves readability and maintainability.

---

## Case Sensitivity

C++ is case-sensitive.

```cpp
int age = 21;
int Age = 25;
```

`age` and `Age` are considered different variables.

---

## Comments

```cpp
// Single-line comment

/*
   Multi-line
   comment
*/
```

Comments are ignored by the compiler and are used to explain code.

---

## Important Points

- Every statement should end with `;`.
    
- `main()` is the entry point of the program.
    
- C++ is case-sensitive.
    
- Use proper indentation for better readability.
    
- Curly braces define code blocks.
    

---

## Common Mistakes

- Forgetting semicolons.
    
- Mismatched curly braces.
    
- Writing `Main()` instead of `main()`.
    
- Assuming C++ is not case-sensitive.
    
- Writing code without proper indentation.
    

---

## Extra Information

### General Structure of a C++ Program

```cpp
#include <header_files>

int main() {
    // Variables
    // Logic
    // Output

    return 0;
}
```

This structure will remain the same for most C++ programs.
