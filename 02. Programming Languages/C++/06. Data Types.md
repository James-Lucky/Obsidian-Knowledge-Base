# C++ 

## What are Data Types?

Data types specify the type of data a variable can store. They determine:

- The size of memory allocated.
    
- The range of values.
    
- The operations that can be performed.
    

---

## Categories of Data Types

1. Fundamental (Built-in)
    
2. Derived
    
3. User-Defined
    

---

## Fundamental Data Types

|Data Type|Description|Typical Size|
|---|---|---|
|`int`|Integer numbers|4 bytes|
|`float`|Decimal numbers|4 bytes|
|`double`|Large decimal numbers|8 bytes|
|`char`|Single character|1 byte|
|`bool`|True or False|1 byte|
|`void`|No value|0 bytes|

---

## Integer (`int`)

Stores whole numbers.

```cpp
int age = 21;
int temperature = -5;
```

---

## Float (`float`)

Stores decimal values with single precision.

```cpp
float price = 99.99f;
```

> [!note]  
> Add `f` at the end of float literals.

---

## Double (`double`)

Stores decimal values with double precision.

```cpp
double pi = 3.1415926535;
```

`double` is generally preferred over `float` due to better precision.

---

## Character (`char`)

Stores a single character enclosed in single quotes.

```cpp
char grade = 'A';
```

---

## Boolean (`bool`)

Stores `true` or `false`.

```cpp
bool isLoggedIn = true;
```

Internally:

- `true` = 1
    
- `false` = 0
    

---

## Void (`void`)

Represents the absence of a value.

```cpp
void display() {
    std::cout << "Hello";
}
```

---

## Derived Data Types

Created from fundamental data types.

- Arrays
    
- Pointers
    
- References
    
- Functions
    

```cpp
int numbers[5];
int* ptr;
int& ref = number;
```

---

## User-Defined Data Types

Created by programmers.

- `struct`
    
- `class`
    
- `enum`
    
- `typedef`
    
- `using`
    

```cpp
struct Student {
    std::string name;
    int age;
};
```

---

## Type Modifiers

Type modifiers change the size or range of data types.

|Modifier|
|---|
|`short`|
|`long`|
|`signed`|
|`unsigned`|

### Examples

```cpp
short int x = 10;
long int y = 100000;
unsigned int z = 50;
```

---

## Type Sizes

```cpp
#include <iostream>

int main() {
    std::cout << sizeof(int) << "\n";
    std::cout << sizeof(double) << "\n";
    std::cout << sizeof(char);
}
```

### Output (Typical)

```text
4
8
1
```

> [!important]  
> Use `sizeof()` to check the size of a data type on your system.

---

## Important Points

- `int` is used for whole numbers.
    
- `double` is preferred over `float` in most cases.
    
- `char` stores only one character.
    
- `bool` stores `true` or `false`.
    
- Data type sizes may vary across systems.
    

---

## Common Mistakes

- Using `"` instead of `'` for `char`.
    
- Forgetting `f` with `float`.
    
- Assuming data type sizes are always the same.
    
- Using `float` when higher precision is required.
    

---

## Extra Information

### Modern Integer Types

C++ provides fixed-width integer types through `<cstdint>`:

```cpp
#include <cstdint>

int8_t a;
int16_t b;
int32_t c;
int64_t d;
```

These are useful when exact memory sizes are required.
