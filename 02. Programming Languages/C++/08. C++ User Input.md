# C++ User Input

## What is User Input?

User input allows a program to receive data from the user during runtime.

In C++, user input is commonly taken using `std::cin`.

---

## Syntax

```cpp
std::cin >> variableName;
```

---

## Example

```cpp
#include <iostream>

int main() {
    int age;

    std::cout << "Enter your age: ";
    std::cin >> age;

    std::cout << "Age: " << age;

    return 0;
}
```

### Output

```text
Enter your age: 21
Age: 21
```

---

## Multiple Inputs

```cpp
int x, y;

std::cin >> x >> y;
```

### Input

```text
10 20
```

### Output

```cpp
std::cout << x << " " << y;
```

```text
10 20
```

---

## Input Different Data Types

```cpp
int age;
double salary;
char grade;

std::cin >> age;
std::cin >> salary;
std::cin >> grade;
```

---

## Reading Strings

### Using `cin`

```cpp
std::string name;

std::cin >> name;
```

### Input

```text
Lucky Sharma
```

### Output

```text
Lucky
```

> [!important]  
> `std::cin` stops reading at the first whitespace.

---

## Using `getline()`

```cpp
std::string fullName;

getline(std::cin, fullName);
```

### Input

```text
Lucky Sharma
```

### Output

```text
Lucky Sharma
```

`getline()` reads the entire line, including spaces.

---

## `cin` and `getline()` Issue

```cpp
int age;
std::string name;

std::cin >> age;
getline(std::cin, name);
```

The `getline()` may be skipped because `cin` leaves a newline character (`\n`) in the input buffer.

### Solution

```cpp
std::cin.ignore();

getline(std::cin, name);
```

---

## Example Program

```cpp
#include <iostream>
#include <string>

int main() {
    std::string name;
    int age;

    std::cout << "Enter your name: ";
    getline(std::cin, name);

    std::cout << "Enter your age: ";
    std::cin >> age;

    std::cout << "\nName: " << name;
    std::cout << "\nAge: " << age;

    return 0;
}
```

---

## Important Points

- `std::cin` is used for user input.
    
- `>>` is called the extraction operator.
    
- `getline()` reads an entire line.
    
- `std::cin.ignore()` is often used before `getline()`.
    

---

## Common Mistakes

- Using `cin` for multi-word strings.
    
- Forgetting `std::cin.ignore()`.
    
- Entering the wrong data type.
    
- Not validating user input.
    

---

## Extra Information

### Input Validation

```cpp
int age;

if (!(std::cin >> age)) {
    std::cout << "Invalid Input!";
}
```

Input validation helps prevent runtime errors and unexpected behavior.
