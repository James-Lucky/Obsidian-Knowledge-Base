# C++ Introduction

## What is C++?

C++ is a general-purpose, high-performance programming language developed by **Bjarne Stroustrup** in 1979 as an extension of the C programming language.

It is used for:

- Software Development
    
- Game Development
    
- Operating Systems
    
- Embedded Systems
    
- Databases
    
- Browsers
    
- Competitive Programming
    
- Data Structures & Algorithms
    

---

## Why Learn C++?

- Fast and efficient.
    
- Provides low-level memory control.
    
- Supports Object-Oriented Programming (OOP).
    
- Widely used in technical interviews and DSA.
    
- Has a powerful Standard Template Library (STL).
    

> [!important]  
> C++ is one of the best languages for learning Data Structures and Algorithms.

## Why is it Called C++?

The `++` operator means increment by one.

```cpp
int x = 5;
x++; // x becomes 6
```

C++ means an improved or enhanced version of C.

## Important Points

- C++ is a compiled language.
    
- C++ supports both procedural and object-oriented programming.
    
- C++17 is commonly used for DSA and interviews.
    
- Modern C++ includes features like smart pointers, lambdas, and multithreading.
    

---

## Common Mistakes

- Assuming C++ is only for competitive programming.
    
- Ignoring pointers and memory management.
    
- Learning OOP before mastering the basics.

## Writing Your First Program

```
#include <iostream>

int main() {
    std::cout << "Hello, World!";
    return 0;
}
```

### Output

```
Hello, World!
```

---

## Understanding the Program

### `#include <iostream>`

- Imports the input/output library.
- Required for `cout` and `cin`.

### `int main()`

- Entry point of every C++ program.
- Execution starts from the `main()` function.

### `std::cout`

- Used to print output to the console.

### `return 0;`

- Indicates successful program execution.

---

## Compilation Process

```
Source Code (.cpp)
        ↓
Compiler (g++)
        ↓
Object Code
        ↓
Executable File
        ↓
Program Runs
```

### Compile and Run

```
g++ main.cpp -o main
./main
```

> [!note]  
> On Windows, use:
> 
> ```
> main.exe
> ```
