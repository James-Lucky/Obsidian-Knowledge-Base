# C++ Structures

## What is a Structure?

A structure (`struct`) is a user-defined data type that groups related variables of different data types under a single name.

It is useful for representing real-world entities.

---

## Syntax

```cpp
struct StructureName {
    dataType member1;
    dataType member2;
};
```

---

## Example

```cpp
struct Student {
    std::string name;
    int age;
    double marks;
};
```

---

## Creating Structure Variables

```cpp
Student s1;
```

---

## Assigning Values

```cpp
s1.name = "Lucky";
s1.age = 21;
s1.marks = 92.5;
```

---

## Accessing Members

Use the dot (`.`) operator.

```cpp
std::cout << s1.name;
std::cout << s1.age;
```

---

## Complete Example

```cpp
#include <iostream>
#include <string>

struct Student {
    std::string name;
    int age;
};

int main() {
    Student s1;

    s1.name = "Lucky";
    s1.age = 21;

    std::cout << s1.name << "\n";
    std::cout << s1.age;
}
```

---

## Structure Initialization

```cpp
Student s1 = {"Lucky", 21};
```

---

## Array of Structures

```cpp
Student students[2] = {
    {"Lucky", 21},
    {"Rahul", 22}
};
```

Accessing:

```cpp
std::cout << students[0].name;
```

---

## Nested Structures

```cpp
struct Address {
    std::string city;
};

struct Student {
    std::string name;
    Address address;
};
```

Usage:

```cpp
Student s;

s.address.city = "Delhi";
```

---

## Structures and Functions

Passing a structure to a function:

```cpp
void display(Student s) {
    std::cout << s.name;
}
```

Passing by reference:

```cpp
void display(const Student& s) {
    std::cout << s.name;
}
```

> [!important]  
> Passing by reference avoids unnecessary copying and is preferred for large structures.

---

## Structure vs Class

|Structure|Class|
|---|---|
|Members are `public` by default|Members are `private` by default|
|Often used for data storage|Used for OOP|

```cpp
struct Student {};
class Student {};
```

---

## Important Points

- Structures are user-defined data types.
    
- Members are public by default.
    
- Use the dot (`.`) operator to access members.
    
- Structures can contain arrays, functions, and other structures.
    

---

## Common Mistakes

- Forgetting the semicolon after the structure definition.
    

```cpp
struct Student {
    int age;
}; // Required
```

- Using `->` instead of `.` for normal objects.
    
- Forgetting to initialize structure members.
    

---

## Extra Information

### Methods Inside Structures

Structures can contain member functions.

```cpp
struct Student {
    std::string name;

    void display() {
        std::cout << name;
    }
};
```

> [!note]  
> In modern C++, `struct` and `class` are almost identical. The primary difference is the default access specifier.
