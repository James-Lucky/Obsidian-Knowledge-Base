
## What are Operators?

Operators are symbols used to perform operations on variables and values.

```cpp
int x = 10 + 5;
```

In the above example:

- `+` is the operator.
    
- `10` and `5` are operands.
    

---

## Types of Operators

1. Arithmetic Operators
    
2. Assignment Operators
    
3. Comparison Operators
    
4. Logical Operators
    
5. Bitwise Operators
    
6. Increment & Decrement Operators
    
7. Ternary Operator
    
8. `sizeof` Operator
    

---

## Arithmetic Operators

|Operator|Description|Example|
|---|---|---|
|`+`|Addition|`a + b`|
|`-`|Subtraction|`a - b`|
|`*`|Multiplication|`a * b`|
|`/`|Division|`a / b`|
|`%`|Modulus|`a % b`|

### Example

```cpp
int a = 10;
int b = 3;

std::cout << a + b; // 13
std::cout << a - b; // 7
std::cout << a * b; // 30
std::cout << a / b; // 3
std::cout << a % b; // 1
```

> [!note]  
> Integer division discards the decimal part.

## Assignment Operators

|Operator|Example|
|---|---|
|`=`|`x = 10`|
|`+=`|`x += 5`|
|`-=`|`x -= 5`|
|`*=`|`x *= 5`|


## Comparison Operators

|Operator|Description|
|---|---|
|`==`|Equal to|
|`!=`|Not equal to|
|`>`|Greater than|
|`<`|Less than|
|`>=`|Greater than or equal to|

## Increment & Decrement Operators

### Increment (`++`)

```cpp
int x = 5;

x++;
++x;
```

### Decrement (`--`)

```cpp
int y = 5;

y--;
--y;
```

### Pre vs Post Increment

```cpp
int x = 5;

std::cout << ++x; // 6
std::cout << x++; // 6
```

|Operator|Action|
|---|---|
|`++x`|Increment first, then use|
|`x++`|Use first, then increment|

## Bitwise Operators

|Operator|Description|
|---|---|
|`&`|AND|
|`|`|
|`^`|XOR|
|`~`|NOT|
|`<<`|Left Shift|
|`>>`|Right Shift|

### Example

```cpp
int x = 5;   // 0101
int y = 3;   // 0011

std::cout << (x & y); // 1
```

---

## Ternary Operator

### Syntax

```cpp
condition ? expression1 : expression2;
```

### Example

```cpp
int age = 18;

std::string result = (age >= 18)
    ? "Adult"
    : "Minor";
```

---

## `sizeof` Operator

Returns the size of a data type or variable in bytes.

```cpp
std::cout << sizeof(int);
```

### Example

```cpp
int x = 10;

std::cout << sizeof(x);
```

## Operator Precedence (High to Low)

|Operator|
|---|
|`()`|
|`++ --`|
|`* / %`|
|`+ -`|
|`< <= > >=`|
|`== !=`|
|`&&`|
|`|
|`=`|

> [!important]  
> Use parentheses `()` when in doubt to improve readability.

---

## Important Points

- Operators perform actions on operands.
    
- `==` checks equality, while `=` assigns values.
    
- `++x` and `x++` behave differently.
    
- `sizeof()` is evaluated at compile time.
    
- Bitwise operators work at the binary level.
    

---

## Common Mistakes

- Using `=` instead of `==`.
    
- Forgetting integer division behavior.
    
- Confusing pre-increment and post-increment.
    
- Misunderstanding operator precedence.
    

---

## Extra Information

### Short-Circuit Evaluation

```cpp
if (x > 0 && y / x > 2) {
    // Code
}
```

If `x > 0` is `false`, the second condition is not evaluated.

This behavior improves performance and prevents errors such as division by zero.
