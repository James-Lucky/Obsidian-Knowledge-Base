# C++ Math

## What is the Math Library?

C++ provides mathematical functions through the `<cmath>` library.

These functions are used for:

- Square roots
    
- Powers
    
- Rounding
    
- Absolute values
    
- Trigonometry
    
- Logarithms
    

---

## Including the Math Library

```cpp
#include <cmath>
```

---

## Common Math Functions

|Function|Description|
|---|---|
|`max()`|Returns the larger value|
|`min()`|Returns the smaller value|
|`sqrt()`|Square root|
|`round()`|Rounds to nearest integer|
|`ceil()`|Rounds up|
|`floor()`|Rounds down|
|`pow()`|Raises a number to a power|
|`abs()`|Absolute value|
|`log()`|Natural logarithm|
|`sin()`|Sine|
|`cos()`|Cosine|
|`tan()`|Tangent|

---

## `max()` and `min()`

```cpp
#include <iostream>

std::cout << std::max(10, 20);
std::cout << std::min(10, 20);
```

### Output

```text
20
10
```

---

## `sqrt()`

Returns the square root of a number.

```cpp
std::cout << sqrt(64);
```

### Output

```text
8
```

---

## `pow()`

Raises a number to a given power.

```cpp
std::cout << pow(2, 3);
```

### Output

```text
8
```

---

## `round()`

Rounds to the nearest integer.

```cpp
std::cout << round(4.6);
```

### Output

```text
5
```

---

## `ceil()`

Always rounds upward.

```cpp
std::cout << ceil(4.1);
```

### Output

```text
5
```

---

## `floor()`

Always rounds downward.

```cpp
std::cout << floor(4.9);
```

### Output

```text
4
```

---

## `abs()`

Returns the absolute (positive) value.

```cpp
std::cout << abs(-25);
```

### Output

```text
25
```

---

## Logarithmic Functions

```cpp
std::cout << log(10);
```

- `log()` → Natural logarithm (base `e`)
    
- `log10()` → Base 10 logarithm
    

---

## Trigonometric Functions

```cpp
std::cout << sin(0);
std::cout << cos(0);
std::cout << tan(0);
```

> [!note]  
> Trigonometric functions use radians, not degrees.

---

## Example Program

```cpp
#include <iostream>
#include <cmath>

int main() {
    std::cout << "Square Root: " << sqrt(25) << "\n";
    std::cout << "Power: " << pow(3, 2) << "\n";
    std::cout << "Maximum: " << std::max(15, 20);

    return 0;
}
```

---

## Important Points

- Include `<cmath>` to use mathematical functions.
    
- `pow()` returns a floating-point value.
    
- Trigonometric functions work with radians.
    
- `ceil()` and `floor()` behave differently.
    

---

## Common Mistakes

- Forgetting to include `<cmath>`.
    
- Assuming `pow()` returns an integer.
    
- Passing degrees to `sin()`, `cos()`, or `tan()`.
    
- Confusing `ceil()` and `floor()`.
    

---

## Extra Information

### Converting Degrees to Radians

```cpp
double degrees = 90;

double radians = degrees * (3.14159265359 / 180);
```

This conversion is necessary when working with trigonometric functions.
