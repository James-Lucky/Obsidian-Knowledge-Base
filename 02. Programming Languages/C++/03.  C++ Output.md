

## What is Output?

Output is the information displayed to the user after a program executes.

In C++, output is typically displayed using `std::cout`.

---

## Syntax

```cpp
std::cout << "Hello, World!";
```

---

## Example

```cpp
#include <iostream>

int main() {
    std::cout << "Welcome to C++!";
    return 0;
}
```

### Output

```text
Welcome to C++!
```

---

## Multiple Outputs

```cpp
std::cout << "Hello ";
std::cout << "World!";
```

### Output

```text
Hello World!
```

---

## Using `<<` Operator

The insertion operator (`<<`) sends data to the output stream.

```cpp
std::cout << "Age: " << 21;
```

### Output

```text
Age: 21
```

---

## Printing Variables

```cpp
int age = 21;

std::cout << "Age: " << age;
```

### Output

```text
Age: 21
```

---

## New Line

### Using `\n`

```cpp
std::cout << "Hello\n";
std::cout << "World";
```

### Output

```text
Hello
World
```

### Using `std::endl`

```cpp
std::cout << "Hello" << std::endl;
std::cout << "World";
```

### Output

```text
Hello
World
```

> [!important]  
> `std::endl` inserts a new line and flushes the output buffer, while `\n` only inserts a new line.

---

## Combining Text and Variables

```cpp
std::string name = "Lucky";
int age = 21;

std::cout << "Name: " << name << "\n";
std::cout << "Age: " << age;
```

### Output

```text
Name: Lucky
Age: 21
```

---

## Escape Sequences

|Escape Sequence|Description|
|---|---|
|`\n`|New Line|
|`\t`|Horizontal Tab|
|`\\`|Backslash|
|`\"`|Double Quote|

### Example

```cpp
std::cout << "Hello\tWorld";
```

---

## Important Points

- `std::cout` is used for output.
    
- `<<` is called the insertion operator.
    
- `\n` is generally faster than `std::endl`.
    
- Multiple values can be printed in a single statement.
    

---

## Common Mistakes

- Forgetting `#include <iostream>`.
    
- Missing `<<` between values.
    
- Using `std::end` instead of `std::endl`.
    
- Forgetting quotation marks around strings.
    

---

## Extra Information

### Best Practice

Prefer using `\n` over `std::endl` unless you specifically need to flush the output buffer.

```cpp
std::cout << "Fast Output\n";
```

This is commonly used in competitive programming and performance-critical applications.
