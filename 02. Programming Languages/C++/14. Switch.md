# C++ Switch

## What is a Switch Statement?

The `switch` statement is used to execute one block of code among many possible options.

It is often used as an alternative to multiple `if-else if` statements when checking a single variable against multiple constant values.

---

## Syntax

```cpp
switch (expression) {
    case value1:
        // Code
        break;

    case value2:
        // Code
        break;

    default:
        // Code
}
```

---

## Example

```cpp
int day = 3;

switch (day) {
    case 1:
        std::cout << "Monday";
        break;

    case 2:
        std::cout << "Tuesday";
        break;

    case 3:
        std::cout << "Wednesday";
        break;

    default:
        std::cout << "Invalid Day";
}
```

### Output

```text
Wednesday
```

---

## How `switch` Works

1. The expression is evaluated.
    
2. The matching `case` is executed.
    
3. `break` exits the `switch` statement.
    
4. If no case matches, `default` executes.
    

---

## The `break` Keyword

Without `break`, execution continues to the next case.

### Example

```cpp
int x = 1;

switch (x) {
    case 1:
        std::cout << "One\n";

    case 2:
        std::cout << "Two\n";
}
```

### Output

```text
One
Two
```

This behavior is called **fall-through**.

> [!important]  
> Always use `break` unless fall-through is intentionally required.

---

## The `default` Keyword

The `default` block executes if no case matches.

```cpp
switch (choice) {
    case 1:
        std::cout << "Option 1";
        break;

    default:
        std::cout << "Invalid Option";
}
```

---

## Multiple Cases

```cpp
char grade = 'B';

switch (grade) {
    case 'A':
    case 'B':
        std::cout << "Passed with Distinction";
        break;

    case 'C':
        std::cout << "Passed";
        break;
}
```

---

## Valid Data Types

`switch` supports:

- `int`
    
- `char`
    
- `enum`
    

Modern C++ also allows:

- `constexpr` values
    

> [!note]  
> `switch` does not work directly with `std::string`.

---

## Example Program

```cpp
#include <iostream>

int main() {
    int month = 5;

    switch (month) {
        case 1:
            std::cout << "January";
            break;

        case 5:
            std::cout << "May";
            break;

        default:
            std::cout << "Unknown";
    }

    return 0;
}
```

---

## Important Points

- `switch` is useful for multiple fixed choices.
    
- `break` prevents fall-through.
    
- `default` is optional but recommended.
    
- Case labels must be constant values.
    

---

## Common Mistakes

- Forgetting `break`.
    
- Using variables inside `case` labels.
    
- Expecting `switch` to work with strings.
    
- Omitting `default`.
    

---

## Extra Information

### When to Use `switch` vs `if-else`

|Use `switch`|Use `if-else`|
|---|---|
|Fixed values|Complex conditions|
|Menus|Ranges (`age > 18`)|
|Enums|Multiple logical expressions|

```cpp
if (marks >= 90) {
    // Better choice
}
```

```cpp
switch (menuChoice) {
    // Better choice
}
```
