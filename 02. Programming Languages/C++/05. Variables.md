# C++ Variables

## What is a Variable?

A variable is a named storage location used to store data in memory. The value of a variable can change during program execution.

---

## Syntax

```cpp
dataType variableName = value;
```

---

## Examples

```cpp
int age = 21;
float height = 5.9;
char grade = 'A';
std::string name = "Lucky";
```

---

## Variable Declaration

Declaring a variable means specifying its type and name.

```cpp
int age;
```

---

## Variable Initialization

Initializing a variable means assigning it a value.

```cpp
int age = 21;
```

You can also assign the value later:

```cpp
int age;
age = 21;
```

---

## Multiple Variables

### Same Type

```cpp
int x = 10, y = 20, z = 30;
```

### Assign Same Value

```cpp
int x, y, z;
x = y = z = 100;
```

---

## Variable Naming Rules

- Can contain letters, digits, and `_`.
    
- Must begin with a letter or `_`.
    
- Cannot start with a number.
    
- Cannot contain spaces.
    
- Cannot use C++ keywords.
    

### Valid Names

```cpp
int age;
int student_age;
int _count;
```

### Invalid Names

```cpp
int 2age;        // Starts with a number
int student age; // Contains space
int class;       // Keyword
```

---

## Naming Conventions

### Camel Case

```cpp
int studentAge;
```

### Snake Case

```cpp
int student_age;
```

> [!note]  
> Camel Case is commonly used in C++ projects.

---

## Scope of Variables

### Local Variable

Declared inside a function or block.

```cpp
void display() {
    int x = 10;
}
```

### Global Variable

Declared outside all functions.

```cpp
int age = 21;

int main() {
    std::cout << age;
}
```

---

## Example Program

```cpp
#include <iostream>

int main() {
    std::string name = "Lucky";
    int age = 21;

    std::cout << "Name: " << name << "\n";
    std::cout << "Age: " << age;

    return 0;
}
```

---

## Important Points

- Variables store data in memory.
    
- Every variable must have a data type.
    
- Variable names are case-sensitive.
    
- Initialize variables whenever possible.
    

---

## Common Mistakes

- Using variables without initialization.
    
- Choosing unclear variable names.
    
- Starting variable names with numbers.
    
- Using reserved keywords as variable names.
    

---

## Extra Information

### Memory Allocation

Each variable occupies memory based on its data type.

|Data Type|Typical Size|
|---|---|
|`int`|4 bytes|
|`float`|4 bytes|
|`double`|8 bytes|
|`char`|1 byte|
|`bool`|1 byte|

> [!important]  
> The exact size of a data type may vary depending on the compiler and system architecture.
