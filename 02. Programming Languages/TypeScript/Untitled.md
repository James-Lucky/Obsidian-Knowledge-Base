
| File Type        | Extension       | When to use                                      |
| ---------------- | --------------- | ------------------------------------------------ |
| React Component  | **`.tsx`** ✅    | Contains JSX/HTML (`<div>`, `<button>`, etc.)    |
| Next.js Page     | **`.tsx`** ✅    | `page.tsx`, `layout.tsx`, `loading.tsx`, etc.    |
| Custom Hook      | `.ts` or `.tsx` | `.ts` if no JSX, `.tsx` if it returns JSX (rare) |
| Utility Function | **`.ts`** ✅     | Helper functions                                 |
| Service/API      | **`.ts`** ✅     | API calls, Axios, fetch                          |
| Types/Interfaces | **`.ts`** ✅     | Interfaces, types                                |
| Constants        | **`.ts`** ✅     | Constant values                                  |
| React Context    | **`.tsx`** ✅    | Contains `<Provider>` JSX                        |
