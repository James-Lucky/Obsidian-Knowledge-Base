Here are the most useful **Obsidian keyboard shortcuts** (Windows/Linux). If you're on Mac, replace **Ctrl** with **Cmd** in most cases.

## File & Navigation

|Action|Shortcut|
|---|---|
|Open Command Palette|`Ctrl + P`|
|Quick Switcher (search notes)|`Ctrl + O`|
|Create New Note|`Ctrl + N`|
|Open New Tab|`Ctrl + T`|
|Close Current Tab|`Ctrl + W`|
|Reopen Closed Tab|`Ctrl + Shift + T`|
|Open Settings|`Ctrl + ,`|
|Go Back|`Alt + Left Arrow`|
|Go Forward|`Alt + Right Arrow`|

## Editing

|Action|Shortcut|
|---|---|

|Italic|`Ctrl + I`|
|Strikethrough|`Alt + Shift + 5`|
|Undo|`Ctrl + Z`|
|Redo|`Ctrl + Y`|
|Find in Current Note|`Ctrl + F`|
|Find and Replace|`Ctrl + H`|
|Delete Current Line|`Ctrl + Shift + K`|
|Duplicate Current Line|`Shift + Alt + Down Arrow`|

|---|---|

## My Recommended Custom Shortcuts for Productivity

Since you're using Obsidian for personal knowledge management and development notes, I'd assign:

- `Alt + N` → New Note
- `Alt + D` → Open Daily Note
- `Alt + T` → Insert Template
- `Alt + J` → Move Note
- `Alt + R` → Rename Note
- `Ctrl + Shift + N` → Create New Folder

## Pro Tip

Press:

`Ctrl + P`

and type **"Hotkeys"** → open **Settings → Hotkeys**.

You can search any command and assign your own shortcuts, which is one of Obsidian's biggest productivity features.

The three shortcuts you'll probably use the most are:

1. `Ctrl + P` → Command Palette
2. `Ctrl + O` → Search Notes
3. `[[` → Create or link notes instantly (not a keyboard shortcut, but essential in Obsidian).
