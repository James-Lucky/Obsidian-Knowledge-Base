# 1. What is Polymorphism?

Polymorphism is derived from two Greek words:

- Poly = Many
    
- Morph = Forms
    

Polymorphism means "one entity having multiple forms."

> "The same function or object behaves differently in different situations."

---

# 2. Why Do We Need Polymorphism?

- Improves flexibility.
    
- Increases code reusability.
    
- Makes programs extensible.
    
- Supports dynamic behavior.
    
- Simplifies code maintenance.
    

---

# 5. Compile-Time Polymorphism

Also known as:

- Static Binding
    
- Early Binding
    

Achieved using:

1. Function Overloading
    
2. Operator Overloading
    
3. Constructor Overloading
    

---

### Example

```cpp
class Demo {
public:
    void show() {
        cout << "No Args";
    }

    void show(int x) {
        cout << x;
    }
};
```

Compiler decides which function to call.

---

# 6. Run-Time Polymorphism

Also known as:

- Dynamic Binding
    
- Late Binding
    

Achieved using:

1. Function Overriding
    
2. Virtual Functions
    

---

### Example

```cpp
class Animal {
public:
    virtual void sound() {
        cout << "Animal";
    }
};

class Dog : public Animal {
public:
    void sound() {
        cout << "Bark";
    }
};
```

Decision is made during execution.

# 9. Compile-Time vs Run-Time

|Compile-Time|Run-Time|
|---|---|
|Early Binding|Late Binding|
|Faster|Slightly Slower|
|Resolved by Compiler|Resolved at Runtime|
|Overloading|Overriding|
|No Virtual Keyword|Uses Virtual Keyword|

---

# 10. Rules

### Compile-Time

- Same function name.
    
- Different parameters.
    
- Compiler resolves calls.
    

### Run-Time

- Requires inheritance.
    
- Uses virtual functions.
    
- Base class pointer/reference required.
    

---

# 11. Complete Example

```cpp
#include <iostream>
using namespace std;

class Animal {
public:
    virtual void sound() {
        cout << "Animal";
    }
};

class Dog : public Animal {
public:
    void sound() {
        cout << "Bark";
    }
};

int main() {
    Animal* ptr;

    Dog d;

    ptr = &d;

    ptr->sound();
}
```

### Output

```text
Bark
```

# 13. Common Mistakes

### Mistake

```cpp
class Animal {
public:
    void sound() {}
};
```

Without `virtual`, runtime polymorphism will not work.

---

### Correct

```cpp
virtual void sound();
```
