# 1. What is `this` Pointer?

`this` is a special pointer available in every non-static member function of a class.

It points to the current object that invokes the member function.

> "`this` stores the address of the current object."

---

# 2. Why Do We Need `this`?

- To refer to the current object.
    
- To resolve naming conflicts.
    
- To return the current object.
    
- Used in method chaining.
    
- Helpful in constructors and operator overloading.
    

---

# 3. Syntax

```cpp
this
```

or

```cpp
this->variableName
```

---

# 4. Basic Example

```cpp
#include <iostream>
using namespace std;

class Student {
public:
    int age;

    void setAge(int age) {
        this->age = age;
    }
};

int main() {
    Student s1;

    s1.setAge(22);

    cout << s1.age;
}
```

### Output

```text
22
```

---

# 5. Why Use `this->`?

Consider:

```cpp
class Student {
public:
    int age;

    void setAge(int age) {
        age = age;
    }
};
```

Here:

```text
Parameter age = Parameter age
```

Object variable remains unchanged.

---

### Correct

```cpp
void setAge(int age) {
    this->age = age;
}
```

```text
this->age
     |
     v
Object's age variable
```

---

# 6. How `this` Works

```text
Student s1;
      |
      v
Address = 1000

s1.setAge(22);

this = 1000
```

# 8. Method Chaining

`this` can return the current object.

```cpp
#include <iostream>
using namespace std;

class Demo {
public:
    Demo& display() {
        cout << "Hello\n";
        return *this;
    }
};

int main() {
    Demo d;

    d.display().display();
}
```

### Output

```text
Hello
Hello
```

---

# 9. Rules

1. Available in non-static member functions.
    
2. Points to the current object.
    
3. Cannot be modified.
    
4. Not available inside static functions.
    
5. Implicitly passed by the compiler.
    

---

# 10. `this` in Constructor

```cpp
class Student {
public:
    int age;

    Student(int age) {
        this->age = age;
    }
};
```

This is one of the most common uses of `this`.

---

# 11. `this` in Memory

```text
Student s1 --> Address 1000

Student s2 --> Address 2000

s1.show()
this = 1000

s2.show()
this = 2000
```

Every object has a different `this` pointer value.

---

# 12. Difference Table

|`this` Pointer|Normal Pointer|
|---|---|
|Implicit|Explicit|
|Created automatically|Created manually|
|Points to current object|Points anywhere|
|Cannot be reassigned|Can be reassigned|

---

# 13. Common Mistakes

### Wrong

```cpp
static void show() {
    cout << this;
}
```

> Error: `this` cannot be used in static functions.

---

### Correct

```cpp
void show() {
    cout << this;
}
```

# 14. Interview Questions

### What is `this` Pointer?

- A special pointer that points to the current object.
    

---

### Can `this` be NULL?

- No (under normal object usage).
    

---

### Can we use `this` in static functions?

- No.
    

---

### What does `this` return?

```cpp
*this
```

- Returns the current object.
    

---

### Is `this` a keyword?

- Yes.
