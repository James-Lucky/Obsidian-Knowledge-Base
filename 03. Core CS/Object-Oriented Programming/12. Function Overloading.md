# 1. What is Function Overloading?

Function Overloading is a feature in C++ where multiple functions have the same name but different parameter lists.

The compiler decides which function to call based on:

- Number of parameters
    
- Type of parameters
    
- Order of parameters
    

> "Same function name, different signatures."

---

# 2. Why Do We Need Function Overloading?

- Improves readability.
    
- Reduces the need for multiple function names.
    
- Makes APIs easier to use.
    
- Supports Compile-Time Polymorphism.
    

---

# 3. Syntax

```cpp
returnType functionName();
returnType functionName(int x);
returnType functionName(int x, int y);
```

---

# 4. Basic Example

```cpp
#include <iostream>
using namespace std;

class Demo {
public:
    void show() {
        cout << "No Arguments\n";
    }

    void show(int x) {
        cout << "Integer: " << x << endl;
    }

    void show(string name) {
        cout << "Name: " << name;
    }
};

int main() {
    Demo d;

    d.show();
    d.show(10);
    d.show("Lucky");
}
```

### Output

```text
No Arguments
Integer: 10
Name: Lucky
```

---

# 5. How Function Overloading Works

```text
show()
   |
   v
show(int)
   |
   v
show(string)
```

Compiler matches the appropriate function signature.

---

# 6. Rules

1. Function names must be the same.
    
2. Parameter lists must be different.
    
3. Return type alone cannot differentiate overloaded functions.
    
4. Compiler performs overload resolution.
    
5. Can be used inside or outside classes.
    

---

# 7. Valid Examples

### Different Number of Parameters

```cpp
void add();
void add(int);
void add(int, int);
```

---

### Different Data Types

```cpp
void print(int);
void print(float);
```

---

### Different Order

```cpp
void fun(int, float);
void fun(float, int);
```

---

# 8. Invalid Example

```cpp
int add(int x);

float add(int x);
```

### Why?

```text
Return type is ignored.
```

Both functions have the same signature.

---

# 9. Function Signature

A Function Signature consists of:

```text
Function Name
+
Parameter Count
+
Parameter Types
+
Parameter Order
```

### Example

```cpp
show(int)
show(float)
show(int, float)
```

---

# 10. Diagram

```text
              show()
                  |
      --------------------------
      |            |           |
      v            v           v
  show()      show(int)   show(string)
```

# 11. Function Overloading vs Function Overriding

|Function Overloading|Function Overriding|
|---|---|
|Same class|Different classes|
|Compile-Time|Run-Time|
|No inheritance needed|Requires inheritance|
|Different parameters|Same parameters|

> Very common interview question.

---

# 12. Common Mistakes

### Wrong

```cpp
void display(int);

int display(int);
```

> Compilation Error

---

### Correct

```cpp
void display(int);

void display(float);
```
