# 1. What is Operator Overloading?

Operator Overloading is a feature in C++ that allows operators to have different meanings depending on the operands.

> "Giving additional functionality to existing operators."

Examples:

```cpp
+
-
*
==
<<
>>
```

---

# 2. Why Do We Need Operator Overloading?

- Improve code readability.
    
- Work with user-defined data types.
    
- Make objects behave like built-in types.
    
- Enable intuitive syntax.
    

---

# 3. Real-Life Example

### Integer Addition

```cpp
10 + 20
```

### String Concatenation

```cpp
"Hello" + "World"
```

### Object Addition

```cpp
obj1 + obj2
```

Operator Overloading allows us to define how `+` behaves for objects.

---

# 4. Syntax

```cpp
returnType operatorOperator(parameters) {
    // Logic
}
```

### Example

```cpp
int operator+(Demo obj);
```

---

# 5. Basic Example

```cpp
#include <iostream>
using namespace std;

class Demo {
public:
    int value;

    Demo(int v) {
        value = v;
    }

    Demo operator+(Demo obj) {
        return Demo(value + obj.value);
    }
};

int main() {
    Demo d1(10);
    Demo d2(20);

    Demo d3 = d1 + d2;

    cout << d3.value;
}
```

### Output

```text
30
```

---

# 6. How It Works

```text
d1 + d2

Compiler Converts:

d1.operator+(d2)
```

# 8. Commonly Overloaded Operators

|Operator|Purpose|
|---|---|
|`+`|Addition|
|`-`|Subtraction|
|`*`|Multiplication|
|`==`|Comparison|
|`[]`|Array Access|
|`()`|Function Call|
|`<<`|Output Stream|
|`>>`|Input Stream|
|`++`|Increment|

---

# 9. Operators That Cannot Be Overloaded

1. `::`
    
2. `.`
    
3. `.*`
    
4. `?:`
    
5. `sizeof`
    

### Mnemonic

```text
:: . .* ?: sizeof
```

> This list is commonly asked in interviews.

---

# 10. Unary Operator Overloading

### Example

```cpp
class Demo {
public:
    int x = 10;

    void operator++() {
        ++x;
    }
};
```

```cpp
Demo d;

++d;
```

---

# 11. Binary Operator Overloading

```cpp
class Demo {
public:
    Demo operator+(Demo obj) {
        return Demo();
    }
};
```

Examples:

```cpp
+
-
*
/
```

---

# 12. Friend Function with Operator Overloading

```cpp
class Demo {
    friend ostream& operator<<(ostream&, Demo&);
};
```

Example:

```cpp
cout << obj;
```

Compiler converts:

```cpp
operator<<(cout, obj);
```

---

# 13. Rules

1. At least one operand must be a user-defined type.
    
2. Cannot create new operators.
    
3. Cannot change operator precedence.
    
4. Cannot change associativity.
    
5. Some operators cannot be overloaded.
    

---

# 14. Member vs Friend Operator

|Member Function|Friend Function|
|---|---|
|Uses `this`|No `this`|
|Inside class|Outside class|
|Simpler|More flexible|

---

# 15. Common Mistakes

### Mistake

```cpp
int operator+(int x, int y);
```

> Error: Both operands are built-in types.

---

### Correct

```cpp
Demo operator+(Demo obj);
```

---

### Mistake

Trying to overload:

```cpp
::
```

> Not possible.
