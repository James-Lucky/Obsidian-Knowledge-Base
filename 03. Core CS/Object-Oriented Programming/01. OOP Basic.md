# 1. What is OOP?

Object-Oriented Programming (OOP) is a programming paradigm that organizes software around **objects and classes** instead of only functions and procedures.

- Procedural Programming → Focuses on functions.
    
- OOP → Focuses on data (objects) and behavior (methods).
    

### Definition

> OOP is a programming model that organizes software design around data (objects) and classes rather than functions and logic.

---

## Why OOP?

- Code Reusability
    
- Better Organization
    
- Easy Maintenance
    
- Security
    
- Scalability
    
- Real-world Modeling

# 3. Class

A class is a blueprint or template used to create objects.

### Example

```cpp
class Student { // The Class
public: // Acess Specifier
    string name; // Attribute
    int age; // Attribute
};
```

Here:

- `Student` → Class
    
- `name`, `age` → Data Members

- The `class` keyword is used to create a class called `MyClass`.
- The `public` keyword is an **access specifier**, which specifies that members (attributes and methods) of the class are accessible from outside the class. You will learn more about [access specifiers](https://www.w3schools.com/cpp/cpp_access_specifiers.asp) later.
- Inside the class, there is an integer variable `myNum` and a string variable `myString`. When variables are declared within a class, they are called **attributes**.
- At last, end the class definition with a semicolon `;`.

---

## Syntax

```cpp
class ClassName {
    // Data Members
    // Member Functions
};
```

## Types of Classes in C++

### 1. Simple Class

```cpp
class Student {};
```

### 2. Abstract Class

Contains at least one pure virtual function.

```cpp
class Animal {
public:
    virtual void sound() = 0;
};
```

### 3. Concrete Class

A class that can create objects.

```cpp
class Car {
public:
    void start() {}
};
```

### 4. Nested Class

Class inside another class.

```cpp
class Outer {
    class Inner {};
};
```

### 5. Template Class

Used for Generic Programming.

```cpp
template <typename T>
class Box {
    T data;
};
```

> For placements, remember: Simple, Abstract, and Template classes are commonly asked.

# 4. Object

An object is an instance of a class.

### Definition

> Object is a real-world entity that occupies memory and has properties and behavior.

---

## Syntax

```cpp
ClassName objectName;
```

---

## Example

```cpp
class Student {
public:
    string name;
    int age;
};

int main() {
    Student s1;

    s1.name = "Lucky";
    s1.age = 22;
}
```

---

## Memory Representation

```cpp
Student s1;
Student s2;
```

- Two different objects.
    
- Separate memory allocation.
    

---

## Access Members

```cpp
objectName.memberName
```

Example:

```cpp
s1.name
s1.age
```

---

# 5. Difference Between Class and Object

|Class|Object|
|---|---|
|Blueprint|Instance|
|No memory allocated|Memory allocated|
|Logical entity|Physical entity|
|Defines properties|Uses properties|

# 6. Basic OOP Program

```cpp
#include <iostream>
using namespace std;

class Student {
public:
    string name;
    int age;

    void display() {
        cout << name << endl;
        cout << age;
    }
};

int main() {
    Student s1;

    s1.name = "Lucky";
    s1.age = 22;

    s1.display();
}
```

Output:

```cpp
Lucky
22
```

# 7. Access Specifiers

|Specifier|Accessible|
|---|---|
|public|Everywhere|
|private|Inside class only|
|protected|Class + Derived classes|

### Example

```cpp
class Demo {
public:
    int a;

private:
    int b;

protected:
    int c;
};
```

> By default, members of a class are private.
