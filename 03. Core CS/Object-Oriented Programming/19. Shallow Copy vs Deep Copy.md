# 1. What is Copying?

Copying means creating a new object using an existing object.

```cpp
Demo d1(10);

Demo d2 = d1;
```

This process uses the Copy Constructor.

---

# 2. Types of Copying

1. Shallow Copy
    
2. Deep Copy

```text
   |

```

---

# 3. What is Shallow Copy?

Shallow Copy copies the values of all members, including pointer addresses.

> "Copies the address, not the actual data."

---

# 4. Example of Shallow Copy

```cpp
class Demo {
public:
    int* ptr;

    Demo(int x) {
        ptr = new int(x);
    }
};
```

```cpp
Demo d1(10);

Demo d2 = d1;
```

---

```text
       d1

         |
```

# 7. Example Problem

```cpp
delete d1.ptr;

delete d2.ptr;
```

### Result

```text
Double Free Error
```

---

# 8. What is Deep Copy?

Deep Copy creates a completely separate copy of the data.

> "Copies data, not addresses."

---

# 9. Deep Copy Example

```cpp
class Demo {
public:
    int* ptr;

    Demo(int x) {
        ptr = new int(x);
    }

    Demo(const Demo& obj) {
        ptr = new int(*obj.ptr);
    }
};
```

---

# 10. Memory Representation

```text
       d1                    d2
        |                     |
        v                     v

    +--------+          +--------+
    | 1000   |          | 2000   |
    +--------+          +--------+
        |                    |
        v                    v
       10                   10
```

Each object owns its own memory.

# 12. Shallow vs Deep Copy

|Shallow Copy|Deep Copy|
|---|---|
|Copies address|Copies data|
|Shared memory|Separate memory|
|Unsafe|Safe|
|Default behavior|Custom implementation|
|May cause double delete|No double delete|

---

# 13. When Does Shallow Copy Occur?

```cpp
Demo d2 = d1;
```

If no custom Copy Constructor exists, the compiler generates one.

---

# 14. Rules

### Shallow Copy

- Copies addresses.
    
- Fast.
    
- Unsafe.
    
- Compiler-generated.
    

### Deep Copy

- Copies actual data.
    
- Safe.
    
- Requires custom Copy Constructor.
    
- Recommended for dynamic memory.
    

---

# 15. Real-Life Example

```text
Photocopy

Shallow:
Two people sharing one book.

Deep:
Each person gets their own book.
```

---

# 16. Common Mistakes

### Mistake

```cpp
Demo(const Demo& obj) {
    ptr = obj.ptr;
}
```

This is still a Shallow Copy.

---

### Correct

```cpp
Demo(const Demo& obj) {
    ptr = new int(*obj.ptr);
}
```

# 17. Interview Questions

---

### Which is safer?
